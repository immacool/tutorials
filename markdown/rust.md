## –ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ

### –°–æ–∑–¥–∞–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö

–î–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –≤ Rust –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `let`:


–í —è–∑—ã–∫–µ Rust —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –¥–≤–∞ —Ç–∏–ø–∞ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö: –∏–∑–º–µ–Ω—è–µ–º—ã–µ –∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–µ. –ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é —è–≤–ª—è—é—Ç—Å—è –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º–∏, –¥–ª—è —Ç–æ–≥–æ —á—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –∏–∑–º–µ–Ω—è–µ–º–æ–π, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `mut` –ø–µ—Ä–µ–¥ –µ—ë –∏–º–µ–Ω–µ–º.

#### –§–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ –≤—ã–≤–æ–¥–∞

–î–ª—è —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –≤—ã–≤–æ–¥–∞ –≤ `println!` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –º–∞–∫—Ä–æ—Å `format!`:

–ú–∞–∫—Ä–æ—Å - —ç—Ç–æ —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è —Å –≤–æ—Å–∫–ª–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º –∑–Ω–∞–∫–æ–º –≤ –∫–æ–Ω—Ü–µ –∏–º–µ–Ω–∏. –ú–∞–∫—Ä–æ—Å—ã –ø–æ–∑–≤–æ–ª—è—é—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏, —á–µ–º –æ–±—ã—á–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏. –û–± —ç—Ç–æ–º –ø–æ–¥—Ä–æ–±–Ω–µ–µ –º–æ–∂–Ω–æ –ø—Ä–æ—á–∏—Ç–∞—Ç—å –≤ [–¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–∏](https://doc.rust-lang.org/book/ch19-06-macros.html) –∏–ª–∏ –≤ –≥–ª–∞–≤–µ [–ú–∞–∫—Ä–æ—Å—ã](#–º–∞–∫—Ä–æ—Å—ã).

### –°–∏—Å—Ç–µ–º–∞ —Ç–∏–ø–æ–≤

–í Rust —Å—É—â–µ—Å—Ç–≤—É–µ—Ç —Å–∏—Å—Ç–µ–º–∞ —Ç–∏–ø–æ–≤, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä—É –ø—Ä–æ–≤–µ—Ä—è—Ç—å —Ç–∏–ø—ã –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –≤–æ –≤—Ä–µ–º—è –∫–æ–º–ø–∏–ª—è—Ü–∏–∏. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –∏–∑–±–µ–∂–∞—Ç—å –æ—à–∏–±–æ–∫ –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –ø—Ä–æ–≥—Ä–∞–º–º—ã.

–í–∞–º –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ —É–∫–∞–∑—ã–≤–∞—Ç—å —Ç–∏–ø –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä Rust —Å–∞–º –æ–ø—Ä–µ–¥–µ–ª–∏—Ç —Ç–∏–ø –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –ø–æ –µ—ë –∑–Ω–∞—á–µ–Ω–∏—é. –ù–æ –º–æ–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å —Ç–∏–ø –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π —è–≤–Ω–æ:

#### –°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö –≤ Rust

- –¶–µ–ª—ã–µ —á–∏—Å–ª–∞: `1`, `2`, `3`, ...

  - `f32` - —á–∏—Å–ª–æ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π 32 –±–∏—Ç–∞
  - `f64` - —á–∏—Å–ª–æ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π 64 –±–∏—Ç–∞

–ú–∞—Å—Å–∏–≤—ã, –≤–µ–∫—Ç–æ—Ä—ã, –∫–æ—Ä—Ç–µ–∂–∏ –∏ –¥—Ä—É–≥–∏–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö –±—É–¥—É—Ç —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω—ã –≤ —Å–ª–µ–¥—É—é—â–µ–π –≥–ª–∞–≤–µ [–ö–æ–ª–ª–µ–∫—Ü–∏–∏](#–∫–æ–ª–ª–µ–∫—Ü–∏–∏).

#### –¶–µ–ª—ã–µ —á–∏—Å–ª–∞

–¶–µ–ª—ã–µ —á–∏—Å–ª–∞ –≤ Rust –º–æ–≥—É—Ç –±—ã—Ç—å –∑–Ω–∞–∫–æ–≤—ã–º–∏ –∏–ª–∏ –±–µ–∑–∑–Ω–∞–∫–æ–≤—ã–º–∏. –ó–Ω–∞–∫–æ–≤—ã–µ —á–∏—Å–ª–∞ –º–æ–≥—É—Ç –±—ã—Ç—å –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º–∏, –∞ –±–µ–∑–∑–Ω–∞–∫–æ–≤—ã–µ - —Ç–æ–ª—å–∫–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–º–∏.

#### –ß–∏—Å–ª–∞ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π

–°—Ç—Ä–æ–∫–∏ –≤ Rust –º–æ–≥—É—Ç –±—ã—Ç—å –¥–≤—É—Ö —Ç–∏–ø–æ–≤: `&str` –∏ `String`.

- `&str` - —ç—Ç–æ —Å—Ç—Ä–æ–∫–∞ –≤ –∫–æ–¥–∏—Ä–æ–≤–∫–µ UTF-8, –∫–æ—Ç–æ—Ä–∞—è –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∞. –≠—Ç–æ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º–∞—è —Å—Ç—Ä–æ–∫–∞.
- `String` - —ç—Ç–æ —Å—Ç—Ä–æ–∫–∞ –≤ –∫–æ–¥–∏—Ä–æ–≤–∫–µ UTF-8, –∫–æ—Ç–æ—Ä–∞—è –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∞. –≠—Ç–æ –∏–∑–º–µ–Ω—è–µ–º–∞—è —Å—Ç—Ä–æ–∫–∞.

```rust

```

–°—Ç—Ä–æ–∫–∏ –≤ Rust —è–≤–ª—è—é—Ç—Å—è –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º–∏ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é. –î–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –∏–∑–º–µ–Ω—è–µ–º–æ–π —Å—Ç—Ä–æ–∫–∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –º–µ—Ç–æ–¥ `String::from`. –ú–µ—Ç–æ–¥ `String::from` —Å–æ–∑–¥–∞–µ—Ç –Ω–æ–≤—É—é —Å—Ç—Ä–æ–∫—É –∏–∑ —Å—Ç—Ä–æ–∫–∏ –≤ –∫–æ–¥–∏—Ä–æ–≤–∫–µ UTF-8. –í –ø—Ä–∏–º–µ—Ä–µ –≤—ã—à–µ —Å–æ–∑–¥–∞–µ—Ç—Å—è –∏–∑–º–µ–Ω—è–µ–º–∞—è —Å—Ç—Ä–æ–∫–∞ `y` –∏–∑ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º–æ–π —Å—Ç—Ä–æ–∫–∏ `x`. –ü–æ–¥—Ä–æ–±–Ω–µ–µ –æ —Ñ—É–Ω–∫—Ü–∏—è—Ö –±—É–¥–µ—Ç —Å–∫–∞–∑–∞–Ω–æ –≤ –≥–ª–∞–≤–µ [–§—É–Ω–∫—Ü–∏–∏](#—Ñ—É–Ω–∫—Ü–∏–∏).

#### –ü—Ä–∏–≤–µ–¥–µ–Ω–∏–µ —Ç–∏–ø–æ–≤

–ü—Ä–∏–≤–µ–¥–µ–Ω–∏–µ —Ç–∏–ø–æ–≤ –≤ Rust –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `as`. –ü—Ä–∏–≤–µ–¥–µ–Ω–∏–µ —Ç–∏–ø–æ–≤ –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ—è–≤–Ω—ã–º –∏–ª–∏ —è–≤–Ω—ã–º.

–ü–æ—Ç–µ—Ä—è –¥–∞–Ω–Ω—ã—Ö –ø—Ä–∏ –ø–µ—Ä–µ–≤–æ–¥–µ —Ç–∏–ø–∞ –º–æ–∂–µ—Ç –ø—Ä–æ–∏–∑–æ–π—Ç–∏, –µ—Å–ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–æ –≤ –Ω–æ–≤–æ–º —Ç–∏–ø–µ. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –º—ã –ø–æ–ø—ã—Ç–∞–µ–º—Å—è –ø—Ä–∏–≤–µ—Å—Ç–∏ —á–∏—Å–ª–æ `500` –∫ —Ç–∏–ø—É `u8`, —Ç–æ –ø–æ—Ç–µ—Ä—è–µ–º –¥–∞–Ω–Ω—ã–µ, —Ç–∞–∫ –∫–∞–∫ `u8` –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å —Ç–æ–ª—å–∫–æ –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç `0` –¥–æ `255`.

```rust

```

–ü—Ä–∏ –ø—Ä–∏–≤–µ–¥–µ–Ω–∏–∏ —Ç–∏–ø–æ–≤ –≤ Rust –Ω–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–∏–≤–µ–¥–µ–Ω–∏—è —Ç–∏–ø–æ–≤. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –º—ã –ø–æ–ø—ã—Ç–∞–µ–º—Å—è —Å–ª–æ–∂–∏—Ç—å —á–∏—Å–ª–æ `5` –∏ —Å—Ç—Ä–æ–∫—É `"5"`, —Ç–æ –ø–æ–ª—É—á–∏–º –æ—à–∏–±–∫—É –∫–æ–º–ø–∏–ª—è—Ü–∏–∏.

```rust

```

–î–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã —Å–ª–æ–∂–∏—Ç—å —á–∏—Å–ª–æ –∏ —Å—Ç—Ä–æ–∫—É, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–∏–≤–µ—Å—Ç–∏ —á–∏—Å–ª–æ –∫ —Å—Ç—Ä–æ–∫–µ. –î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Ñ—É–Ω–∫—Ü–∏—è `to_string`.

```rust

```

–í—ã–≤–æ–¥:

```text

```

–ò –Ω–∞–æ–±–æ—Ä–æ—Ç, –¥–ª—è —á–∏—Å–ª–µ–Ω–Ω–æ–≥–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–∏–≤–µ—Å—Ç–∏ —Å—Ç—Ä–æ–∫—É –∫ —á–∏—Å–ª—É. –î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Ñ—É–Ω–∫—Ü–∏—è `parse`.

–†–∞–∑–±–µ—Ä–µ–º —Å—Ç—Ä–æ–∫—É —Å –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ–º —Ç–∏–ø–æ–≤ –ø–æ —á–∞—Å—Ç—è–º.

- `y.parse::<i32>()` - –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏ `y` –∫ —Ç–∏–ø—É `i32`. –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç `Result`, –∫–æ—Ç–æ—Ä—ã–π —Å–æ–¥–µ—Ä–∂–∏—Ç –ª–∏–±–æ —á–∏—Å–ª–æ, –ª–∏–±–æ –æ—à–∏–±–∫—É. –î–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å —á–∏—Å–ª–æ, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –≤—ã–∑–≤–∞—Ç—å –º–µ—Ç–æ–¥ `unwrap`. –ï—Å–ª–∏ –≤ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –ø—Ä–æ–∏–∑–æ—à–ª–∞ –æ—à–∏–±–∫–∞, —Ç–æ –ø—Ä–æ–≥—Ä–∞–º–º–∞ –∑–∞–≤–µ—Ä—à–∏—Ç—Å—è —Å –æ—à–∏–±–∫–æ–π.
- `.unwrap()` - –≤—ã–∑–æ–≤ –º–µ—Ç–æ–¥–∞ `unwrap` –¥–ª—è `Result`. –ï—Å–ª–∏ –≤ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –ø—Ä–æ–∏–∑–æ—à–ª–∞ –æ—à–∏–±–∫–∞, —Ç–æ –ø—Ä–æ–≥—Ä–∞–º–º–∞ –∑–∞–≤–µ—Ä—à–∏—Ç—Å—è —Å –æ—à–∏–±–∫–æ–π.

### –ö–æ–ª–ª–µ–∫—Ü–∏–∏

–ö–æ–ª–ª–µ–∫—Ü–∏–∏ - —ç—Ç–æ —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–∑–≤–æ–ª—è—é—Ç —Ö—Ä–∞–Ω–∏—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –∑–Ω–∞—á–µ–Ω–∏–π –≤ –æ–¥–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π. –í Rust –µ—Å—Ç—å —Ç—Ä–∏ –æ—Å–Ω–æ–≤–Ω—ã—Ö —Ç–∏–ø–∞ –∫–æ–ª–ª–µ–∫—Ü–∏–π: –≤–µ–∫—Ç–æ—Ä, —Å—Ç—Ä–æ–∫–∞ –∏ —Ö—ç—à-–∫–∞—Ä—Ç–∞ (—Å–ª–æ–≤–∞—Ä—å, —Ö—ç—à-—Ç–∞–±–ª–∏—Ü–∞).

- –í–µ–∫—Ç–æ—Ä - —ç—Ç–æ –∏–∑–º–µ–Ω—è–µ–º–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è —Ö—Ä–∞–Ω–∏—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –æ–¥–Ω–æ–≥–æ —Ç–∏–ø–∞. –í–µ–∫—Ç–æ—Ä –º–æ–∂–Ω–æ –∏–∑–º–µ–Ω—è—Ç—å, –¥–æ–±–∞–≤–ª—è—Ç—å –∏ —É–¥–∞–ª—è—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã. –í–µ–∫—Ç–æ—Ä –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `vec!`.
- –°—Ç—Ä–æ–∫–∞ - —ç—Ç–æ –∏–∑–º–µ–Ω—è–µ–º–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è —Ö—Ä–∞–Ω–∏—Ç —Å–∏–º–≤–æ–ª—ã. –°—Ç—Ä–æ–∫–∞ –º–æ–∂–Ω–æ –∏–∑–º–µ–Ω—è—Ç—å, –¥–æ–±–∞–≤–ª—è—Ç—å –∏ —É–¥–∞–ª—è—Ç—å —Å–∏–º–≤–æ–ª—ã. –°—Ç—Ä–æ–∫–∞ –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `String::new`.
- –•—ç—à-–∫–∞—Ä—Ç–∞ - —ç—Ç–æ –∏–∑–º–µ–Ω—è–µ–º–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è —Ö—Ä–∞–Ω–∏—Ç –ø–∞—Ä—ã –∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ. –í —Ö—ç—à-–∫–∞—Ä—Ç–µ –º–æ–∂–Ω–æ –∏–∑–º–µ–Ω—è—Ç—å, –¥–æ–±–∞–≤–ª—è—Ç—å –∏ —É–¥–∞–ª—è—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã. –•—ç—à-–∫–∞—Ä—Ç—É –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `HashMap::new`.
- –ú–∞—Å—Å–∏–≤ - —ç—Ç–æ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è —Ö—Ä–∞–Ω–∏—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –æ–¥–Ω–æ–≥–æ —Ç–∏–ø–∞. –ú–∞—Å—Å–∏–≤ –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `array!`.

## –£—Å–ª–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

### –ú–∞–∫—Ä–æ—Å—ã

–¢–µ–ø–µ—Ä—å –Ω–∞–ø–∏—à–µ–º –ø—Ä–æ—Å—Ç–æ–π –º–∞–∫—Ä–æ—Å —Å–∞–º–∏:

–¢–∞–∫ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø–æ—Ç–æ–º—É, —á—Ç–æ –º–∞–∫—Ä–æ—Å—ã - —ç—Ç–æ —Ñ—É–Ω–∫—Ü–∏–∏, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–∏–Ω–∏–º–∞—é—Ç –∫–æ–¥ –≤ –∫–∞—á–µ—Å—Ç–≤–µ –∞—Ä–≥—É–º–µ–Ω—Ç–∞ –∏ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –∫–æ–¥. –í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ –º–∞–∫—Ä–æ—Å `say_hello!` –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –ø—É—Å—Ç–æ–π –Ω–∞–±–æ—Ä –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ –∏ –∑–∞–º–µ—â–∞–µ—Ç —Å–æ–±–æ–π –≤—ã–∑–æ–≤ —Ñ—É–Ω–∫—Ü–∏–∏ `println!`, –æ–±–æ–∑–Ω–∞—á–µ–Ω–Ω—ã–π –≤ —Ç–µ–ª–µ –º–∞–∫—Ä–æ—Å–∞ –ø–æ—Å–ª–µ —Å–∏–º–≤–æ–ª–∞ `=>`.

### –ê—Ä–≥—É–º–µ–Ω—Ç—ã –º–∞–∫—Ä–æ—Å–æ–≤

–ú–∞–∫—Ä–æ—Å—ã –º–æ–≥—É—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å –∞—Ä–≥—É–º–µ–Ω—Ç—ã. –í–æ—Ç –ø—Ä–∏–º–µ—Ä –º–∞–∫—Ä–æ—Å–∞, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –æ–¥–∏–Ω –∞—Ä–≥—É–º–µ–Ω—Ç:

–≠—Ç–æ—Ç –∫–æ–¥ –Ω–∞ —ç—Ç–∞–ø–µ –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ –ø—Ä–µ–≤—Ä–∞—Ç–∏—Ç—Å—è –≤:

–í —Ç–µ–ª–µ –º–∞–∫—Ä–æ—Å–∞ –º—ã –º–æ–∂–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é `$name`, –∫–æ—Ç–æ—Ä–∞—è –±—É–¥–µ—Ç –∑–∞–º–µ—â–µ–Ω–∞ –Ω–∞ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ä–≥—É–º–µ–Ω—Ç–∞ `name`. –í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ –∞—Ä–≥—É–º–µ–Ω—Ç `name` - —ç—Ç–æ –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä, –ø–æ—ç—Ç–æ–º—É –º—ã —É–∫–∞–∑—ã–≤–∞–µ–º —ç—Ç–æ –≤ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–∏ –º–∞–∫—Ä–æ—Å–∞ —Å –ø–æ–º–æ—â—å—é `:ident`. –í—Å–µ–≥–æ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç 7 —Ç–∏–ø–æ–≤ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤:

stmt: a statement
pat: a pattern

- `item` - —ç–ª–µ–º–µ–Ω—Ç, –Ω–∞–ø—Ä–∏–º–µ—Ä —Ñ—É–Ω–∫—Ü–∏—è, —Å—Ç—Ä—É–∫—Ç—É—Ä–∞, –º–æ–¥—É–ª—å –∏ —Ç.–¥.
- `block` - –±–ª–æ–∫ (—Ç–æ –µ—Å—Ç—å –±–ª–æ–∫ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –∏/–∏–ª–∏ –≤—ã—Ä–∞–∂–µ–Ω–∏–π, –∑–∞–∫–ª—é—á–µ–Ω–Ω—ã—Ö –≤ —Ñ–∏–≥—É—Ä–Ω—ã–µ —Å–∫–æ–±–∫–∏)

–ü—Ä–∏–º–µ—Ä —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –¥—Ä—É–≥–∏—Ö —Ç–∏–ø–æ–≤, –∏ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ–º –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤:

### N –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤

–ú–∞–∫—Ä–æ—Å—ã –º–æ–≥—É—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å –ª—é–±–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤. –í–æ—Ç –ø—Ä–∏–º–µ—Ä –º–∞–∫—Ä–æ—Å–∞, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –ª—é–±–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤:

### Some

–¢–∏–ø `Option` –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –ª–∏–±–æ `Some`, –ª–∏–±–æ `None`. –¢–∏–ø `Some` —Å–æ–¥–µ—Ä–∂–∏—Ç –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –±—ã–ª–æ –≤–æ–∑–≤—Ä–∞—â–µ–Ω–æ —Ñ—É–Ω–∫—Ü–∏–µ–π, –∞ —Ç–∏–ø `None` –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Ñ—É–Ω–∫—Ü–∏—è –Ω–µ –≤–µ—Ä–Ω—É–ª–∞ –Ω–∏–∫–∞–∫–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è.
